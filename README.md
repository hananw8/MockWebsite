# Bookstore Website
A multi-page bookstore website built using HTML, CSS, and JavaScript.
This project was created as a front-end learning exercise to practise layout design, 
form validation, API usage, and file organisation.
# Pages Included
- Home (Book catalogue)
- About page
- Contact page
- Payment page (Demo)
- Success page
# Features
- Shared navigation menu across all pages
- Book layout with images and descriptions
- Card form validation using JavaScript
- API request using JavaScript Fetch
- LocalStorage used to display card details on success page
- External CSS styling for layout and design consistency
# Technologies Used
- HTML5  
- CSS3  
- JavaScript 
# How to Run
1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. Navigate through the website using the menu.
4. Click "Pay" on any book to view the demo checkout page.
# Payment System Disclaimer
The payment system uses a public test API for demonstration purposes only.
No real payments are processed.  
No card details are sent or stored on any server.
This functionality exists purely to demonstrate front-end validation
and API handling using JavaScript.
